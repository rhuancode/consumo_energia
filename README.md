# Calculadora de Consumo Elétrico Inteligente

![GitHub](https://img.shields.io/badge/GitHub-181717?style=for-the-badge&logo=github&logoColor=white)
![Status](https://img.shields.io/badge/Status-Ativo-success?style=for-the-badge)

Feito em Python ![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white) 
para ajudar usuários a estimar quanto um aparelho gasta de energia 
![Energia](https://img.shields.io/badge/Energia-Elétrica-yellow?style=for-the-badge&logo=lightning&logoColor=black) 
elétrica por mês, com base em dados simples de uso.

O programa solicita o **nome do aparelho** ![Aparelho](https://img.shields.io/badge/Aparelho-Seleção-blueviolet?style=for-the-badge&logo=plug&logoColor=white), 
**potência do aparelho** ![Potência](https://img.shields.io/badge/Potência-Watts-orange?style=for-the-badge&logo=flash&logoColor=white) 
e **tempo médio de uso** ![Tempo](https://img.shields.io/badge/Tempo-Uso-informational?style=for-the-badge&logo=clockify&logoColor=white), 
calcula o consumo mensal, custo estimado e mostra o consumo e o custo mensal estimados.
